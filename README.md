# 🛵 Cleidson Delivery - Landing Page Premium

![Banner do Projeto](./Projeto001.jpg)

## 📋 Sobre o Projeto
Landing Page desenvolvida para a **Cleidson Delivery**, uma empresa de logística e entregas rápidas baseada em Natal/RN. O objetivo principal foi criar uma presença digital robusta, transmitindo agilidade, segurança e profissionalismo através de uma interface moderna e de alto desempenho.

---

## 🚀 Funcionalidades e Diferenciais
* **Design Responsivo:** Adaptado para Desktop, Tablet e Mobile.
* **Tema Dark/Light:** Alternância dinâmica de cores conforme a preferência do usuário.
* **Animações de Scroll:** Efeito de revelação (Scroll Reveal) para uma navegação fluida.
* **Otimização Visual:** Imagens com estética Chiaroscuro (luz e sombra) para um ar luxuoso.
* **Integração Direta:** Botões de CTA direcionando diretamente para o WhatsApp da empresa.

---

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estrutura semântica para SEO.
* **CSS3 (Variáveis & Flexbox/Grid):** Estilização avançada e gerenciamento de temas.
* **JavaScript (ES6):** Lógica para troca de tema e manipulação da DOM.
* **Boxicons:** Biblioteca de ícones modernos.
* **Google Fonts:** Tipografia focada em legibilidade (Poppins).

## 🧠 Métodos Abordados
* **Mobile-First:** Priorização da experiência em dispositivos móveis.
* **Clean Code:** Organização de CSS através de variáveis e comentários estruturados.
* **UI/UX Design:** Foco em hierarquia visual e contraste (estética Preto & Dourado).
* **IA-Assisted Development:** Utilização de Inteligência Artificial para refinamento de código e geração de assets visuais exclusivos.
